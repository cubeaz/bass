
# GrooveMaster Pro Bass

这是一个由ai生成的交互式的四弦贝斯模拟器网页应用，所有的效果的真实性均有待考量，主要为图一乐性质。

## 功能特性
- **真实物理建模**：模拟琴弦振动、品格交互和拨弦动态。
- **自定义配置**：支持切换 J-J, P-J, MM-J 拾音器配置。
- **视觉风格**：一键切换 Classic (经典), Modern (现代), Vintage (复古), Cyber (赛博) 四种皮肤。
- **专业音频链**：内置 12 款效果器（压缩、八度、法兹、过载、合唱、延迟、混响、泰坦前级等）和 10 款经典箱头模拟。

## 如何打包 (Packaging)

本项目使用 [Vite](https://vitejs.dev/) 构建。要将应用打包为可发布的静态文件，请按照以下步骤操作：

### 1. 安装依赖
在项目根目录下打开终端，运行：
```bash
npm install
```

### 2. 运行打包命令
执行构建脚本以生成生产环境文件：
```bash
npm run build
```

### 3. 获取成品
构建完成后，项目目录中会出现一个 **`dist`** 文件夹。
- 这个 `dist` 文件夹就是“打包”好的所有内容。
- 里面包含优化过的 HTML、JavaScript 和 CSS 文件。
- 由于配置了相对路径 (`base: './'`)，您可以直接用浏览器打开 `dist/index.html` 进行预览，或者将其上传到任何静态网站托管服务（如 GitHub Pages, Vercel, Netlify）。

## 本地开发
如果在打包前想在本地调试代码：
```bash
npm run dev
```
